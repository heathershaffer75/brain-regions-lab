# 🧠 Brain Regions Lab

## Interactive Recovery Neuroscience

Brain Regions Lab is an interactive psychoeducation tool designed to help patients explore how trauma, attachment, chronic stress, and substance use can affect different regions of the brain.

This project blends:

* Recovery education
* Trauma-informed care
* Neuroscience
* Emotional regulation concepts
* Patient-friendly psychoeducation
* Interactive learning

The goal is to reduce shame, increase insight, and help patients better understand their thoughts, emotions, cravings, reactions, and nervous system responses.

---

# 🌱 Core Therapeutic Message

> “Your brain adapted to survive.”

And:

> “Healing and change are possible.”

This project is designed to help patients understand:

* Why they react the way they do
* Why cravings feel intense
* Why trauma responses happen automatically
* How attachment affects the nervous system
* Why emotional regulation can feel difficult
* How recovery supports brain healing over time

---

# 🎯 Features

## 🧠 Interactive Brain Map

Patients can click brain regions to learn about:

* Primary functions
* Trauma impact
* Attachment impact
* Substance use impact
* Common lived experiences
* Healing and recovery supports

Current brain regions include:

* Amygdala
* Prefrontal Cortex
* Hippocampus
* Insula
* Cerebellum
* Nucleus Accumbens
* Brainstem

---

## 🔎 Symptom Explorer

Patients can select experiences such as:

* Panic
* Cravings
* Hypervigilance
* Emotional flooding
* Shame spirals
* Brain fog
* Numbness
* Impulsivity

The app then explores:

* Possible brain systems involved
* Nervous system explanations
* Reflection prompts

---

## 🎮 Learning Modes

Different discussion lenses:

* General

These modes help facilitators adapt the discussion for different group populations.

This project was designed to feel:

* Emotionally safe
* Engaging

* Non-judgmental
